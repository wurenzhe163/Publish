## ğŸ¯ Research Projects

### 1. **DEM-Based Radar Incidence Angle Tracking for Distortion Analysis Without Orbital Data**
A methodology for geometric distortion detection in SAR imagery using DEM-based approaches, eliminating the need for complex orbital state vector information.

### 2. **Dual-Orbit Sentinel-1 SAR Fusion for Geological Lineament Extraction**  
A dual-orbit fusion approach that combines ascending and descending SAR acquisitions to improve geological feature extraction in mountainous terrain.
